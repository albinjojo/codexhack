# BinBuddy Kochi ♻️

**Find the right bin, before waste reaches the street.**

BinBuddy Kochi helps people find suitable waste bins while travelling through Kochi. A user selects where they are starting from, where they are going, and the type of waste they have. The app then shows suitable bins near their route.

## The problem

People often carry small waste like plastic bottles, food wrappers, or paper while travelling. They may not know where to dispose of it properly, so waste can end up on streets, footpaths, or canals.

BinBuddy helps make responsible disposal easier.

## Features

- Select a starting location and destination in Kochi
- Select waste type: plastic, food waste, paper, or general waste
- View a route on an interactive map
- Find suitable waste bins near the route
- See bin status: available, full, or unknown
- See verified and community-reported bin information
- View prototype bin locations across Kochi

## How it works

1. Select your starting point.
2. Select your destination.
3. Select the waste type.
4. Click **Find the right bin**.
5. View bins that match your waste near your route.

## Tech stack

- Next.js
- React
- Tailwind CSS
- Leaflet and React Leaflet
- OpenStreetMap
- OSRM routing
- Local JSON data for prototype waste-bin locations
